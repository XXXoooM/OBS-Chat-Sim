# 🚀 OBS 直播间弹幕模拟器 (Pro v4.0)

一款专为直播场景打造的**超轻量、单文件**弹幕互动模拟工具。基于 React 18 与 Framer Motion 构建，支持原生透明通道、动态加权随机语料及专业级视觉动效。

---

## ✨ 核心特性 (v4.0 更新)

- **💎 原生透明通道 (Native Alpha)**：彻底抛弃“色度键”抠图，文字边缘丝滑无锯齿，支持 100% 还原的半透明胶囊气泡。
- **🛡️ 阶梯式勋章体系**：
    - **等级勋章**：内置四阶精美 SVG 图标（1-6级单星、7-12级星芒、13-19级三星汇聚、20-30级尊享星芒）。
    - **粉丝勋章**：胶囊型“粉丝”标牌，支持随字号自动缩放。
- **🎛️ 极客黑毛玻璃设置面板**：
    - **Glassmorphism 视觉**：20px 级深度毛玻璃滤镜与细腻高光边缘。
    - **双维度分区**：分为“基础视觉”与“模拟引擎”两大模块，配置逻辑更清晰。
- **🛍️ 「正在购买」联动**：
    - **横向弹出通知**：高还原度购物袋图标，支持原地渐隐退出。
    - **buy 专用弹幕**：自动拼接金色“1号商品”后缀。
- **🎭 丝绸级动效 (Framer Motion)**：
    - **Layout 自动补位**：弹幕消失后，下方弹幕以平滑弹簧物理效果向上滚动。
    - **多维动画**：进入时缩放淡入，退出时向中心收缩消失，拒绝生硬跳变。
- **🎲 全球化随机引擎**：
    - **800+ 跨国字符**：涵盖多国语言、特殊符号及流行网名元素，营造真实公屏氛围。

## 🛠️ OBS 使用指南

为了获得最佳视觉效果，请务必开启 OBS 的**原生透明**支持。

### 1. 模拟器配置
- 打开 `index.html`，双击背景或点击右上角 **「系统设置 (CONFIG)」**。
- 在「基础视觉」分区，确保「画布背景」选择为 **「透明模式」**（此时会显示蓝色标签：*原生透明通道已开启*）。

### 2. OBS 核心设置 (必看)
1. 在 OBS 中添加 **「浏览器」** 源。
2. 勾选 **「本地文件」**，选择 `index.html`。
3. **关键操作**：务必勾选底部的 **「使背景透明 (Make background transparent)」**。
4. **进阶推荐**：建议勾选 **「使用自定义帧率」** 和 **「关闭时释放资源」** 以获得最佳性能。
5. **清理滤镜**：请勿添加任何“色度键”或“滤镜”，原生透明不需要抠图。

### 🔍 为什么我的背景不是透明的？
如果你在 OBS 中仍然看到黑底或绿底，请检查以下几点：
- **自定义 CSS**：确保源属性中的“自定义 CSS”区域为空，或者没有 `body { background: ... }` 的覆盖。
- **强制刷新**：点击属性面板中的 **「刷新当前页面的缓存」**。
- **三重技术保障**：本项目已在底层锁定透明状态：
    - **CSS 层面**：`body` 样式中硬编码了 `background-color: transparent;`。
    - **HTML 层面**：`body` 标签强制携带 `bg-transparent` 类。
    - **默认设置**：系统初始 `bgColor` 变量即为 `'transparent'`。

## ⚙️ 设置项要点

| 模块 | 配置项 | 说明 | 建议值 |
| :--- | :--- | :--- | :--- |
| **基础视觉** | **字体大小** | 决定气泡与图标的整体比例，全自动响应式。 | 15-18px |
| **基础视觉** | **显示区域宽度** | 调整弹幕区的横向占比。 | 350px |
| **基础视觉** | **顶部渐隐深度** | 弹幕滚到顶部时的消失过渡长度。 | 15% |
| **模拟引擎** | **频率触发逻辑** | **固定频率** (匀速) 或 **随机波动** (模拟真实流量浪涌)。 | 随机波动 |
| **模拟引擎** | **波动周期** | 设置流量从爆发到平滑的时间跨度。 | 30 - 120s |
| **模拟引擎** | **消息密度** | 单个波动周期内发送的消息数量范围。 | 10 - 30 |
| **模拟引擎** | **自动销毁** | 弹幕在屏幕上留存的时间。 | 15s |

## 📂 开发与自定义

本项目采用**零依赖单文件**架构（通过 CDN 引入库），你只需一个 `index.html` 即可运行。

- **自定义语料**：在代码中查找 `MESSAGE_GROUPS` 数组，直接修改 `phrases` 即可。
- **商品名称**：修改 `productText` 变量或在设置面板中直接调整（后续计划加入）。
- **持久化**：所有配置均实时保存至浏览器 `LocalStorage`，关闭网页不丢失。

## ⚠️ 注意事项

1. **浏览器源缓存**：在 OBS 中修改代码后，需点击浏览器源属性中的 **「刷新当前页面的缓存」** 才能生效。
2. **硬件加速**：确保 OBS 设置中开启了“浏览器源硬件加速”，以获得最流畅的 60 帧动画。
3. **字体支持**：默认优先调用 `Microsoft YaHei UI Light`，建议在安装了该字体的 Windows 系统上运行。
